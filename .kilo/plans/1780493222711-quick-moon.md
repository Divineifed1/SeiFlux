# SeiFlux Plan

## 1. Navbar and Footer
- State: Already implemented and included in app/layout.tsx by existing `app/components/Navbar` and `app/components/Footer`. No structural work needed unless style changes are requested later.

## 2. Reward System Change: Points Instead of SEI
Current code rewards display in SEI coin amounts, e.g. `1000 SEI`, `50 SEI`, `2500 SEI`. Replace with difficulty-based points:
- Easy: 50 points/points display
- Medium: 75 points
- Hard: 100 points

Files to update:
- `app/projects/[id]/page.tsx`: reward display text ("50 SEI" -> "50 Points", etc.)
- `app/profile/[id]/page.tsx`: rewards displayed in history and summary
- `app/admin/page.tsx`: total rewards pool display
- `app/projects/page.tsx`: reward pool display
- Backend API if needed

Note: Keep reward pool in SEI for owner-to-distribute concept, but display task rewards in points.

## 3. Monthly Waves Feature
Add a **Monthly Waves** feature where each month features a themed 7-day wave of activities/events.

Wave Structure:
- Waves happen once per month
- Each wave lasts exactly 7 days
- Themed challenge periods (e.g., "Security Wave", "UI Wave", "DeFi Wave")
- Special tasks, bonus points, and leaderboards during waves
- Wave countdown timer on dashboard and landing page
- Wave calendar/schedule showing upcoming waves

Pages to build:
- `/waves` page listing current/upcoming/past waves
- Wave detail page with wave-specific tasks
- Wave leaderboard showing top contributors for that wave
- Admin page to create/edit waves and attach wave-specific tasks

Backend endpoints:
- `GET /waves` - list waves
- `GET /waves/:id` - wave detail
- `POST /waves` - create wave (admin)
- `PATCH /waves/:id` - update wave (admin)

UI Components:
- Wave countdown banner/hero
- Wave progress bar
- Wave task cards with bonus indicators
- Wave leaderboard table
- Wave calendar widget

Implementation order:
1. Build backend Entities and API for Waves
2. Build frontend `/waves` page and wave detail routes
3. Integrate wave countdown into navbar/hero
4. Add wave tasks to existing project task system
5. Build wave leaderboard
